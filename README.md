# block-stacker# 🧱 Tetris Emulator (Windows)

A lightweight native **Tetris emulator** built using **C# and Windows Forms**. This project recreates the classic Tetris experience in a self-contained `.exe` you can run directly on Windows.

---

## 🎮 Features

- Classic Tetris block mechanics and rotations
- Line-clearing logic with gravity
- Smooth keyboard controls
- Minimal, responsive UI
- Built with .NET Framework – runs natively as `.exe`

---

## 🕹️ Controls

| Key         | Action             |
|-------------|--------------------|
| ← / →       | Move left / right  |
| ↓           | Soft drop          |
| ↑           | Rotate piece       |
| Spacebar    | Hard drop (instant)|
